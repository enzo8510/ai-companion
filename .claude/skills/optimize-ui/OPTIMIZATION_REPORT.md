## 實施的優化

### 1. **間距標準化** ✅
- **建立 8px 基礎單位的統一刻度**
  - `--space-xs: 4px` (最小間距)
  - `--space-sm: 8px` (小間距)
  - `--space-md: 12px` (中間距)
  - `--space-lg: 16px` (標準間距)
  - `--space-xl: 24px` (大間距)
  - `--space-2xl: 32px` (更大間距)
  - `--space-3xl: 48px` (最大間距)

- **應用範圍**：
  - 主頁佈局（.home-wrap, .home-avatar-ring, .home-actions）
  - 卡片間距（.kb-card, .mem-card, .jr-card, .home-qa-card）
  - 聊天界面（.chat-thread, .bubble）
  - 管理面板（.admin-body, .panel）
  - 星形視圖（.star-stats, .star-actions）
  - 社交視圖（.social-post）

### 2. **色彩對比度改進** ✅
- **改進次要文本顏色**：
  - `--ink-soft` 從 `#8891A4` → `#7A8E9F`
  - 提高了 WCAG AA 標準的合規性
  - 改進了在淺色和深色主題上的可讀性

### 3. **排版層級改進** ✅
- **主頁標題**：
  - `.home-name` 字體大小 34px → 36px
  - 行高設置為 1.2（更緊湊）
  - 底部邊距 7px → `--space-md` (12px)

- **標語**：
  - 字體大小 12.5px → 13px
  - 行高 1.7 → 1.8（改進可讀性）
  - 字母間距 `.06em` → `.04em`（更平衡）

- **氣泡**：
  - 字體大小 14.5px → 14px（更標準）
  - 行高 1.65 → 1.6（更一致）

- **面板標題**：
  - 字體大小 16px（保持）
  - 底部邊距 4px → `--space-md` (12px)
  - 字體權重設置為 700（明確粗體）

### 4. **觸控目標大小改進** ✅
- **icon-btn**：
  - 填充 5px 8px → `--space-md` `--space-lg`
  - 最小寬度/高度：44px（觸控標準）
  - 添加 flexbox 居中

- **按鈕**：
  - 填充 9px 16px → `--space-md` `--space-lg`
  - 最小高度：44px
  - 字體大小 13px → 14px

### 5. **視覺層級簡化** ✅
- 統一所有組件的強調色使用
- 確保清晰的主要/次要/三級動作區分
- 改進按鈕懸停狀態

### 6. **空白空間增強** ✅
- 增加卡片之間的邊距
- 改進部分內邊距
- 更好的垂直節奏感
