人格設定

這裡決定卄珩「是誰、怎麼說話」。下面的設定會在每次對話時提供給 AI，讓他保持人設一致。

事件

上傳或貼上角色設定、世界觀、人物關係等資料。每筆事件最多 2000 字，最多 30 筆。這些內容會一起提供給 AI，讓卄珩更貼合設定。

⚡ Token 容量 計算中
0 / 60,000 字 ~0 tokens

記憶體

由角色依照每 10–20 則對話自動生成的記憶內容（標示「自動記憶」），越新的記憶越被重視。最多儲存 50 則，你也可以手動新增、修改或刪除。每則最多 80 字。

🧠 記憶容量 計算中
0 / 50 則・0 / 4,000 字 ~0 tokens

生活日誌

由你自行新增與編輯的生活日誌，記錄每天的點滴。可以新增、修改或刪除每一頁。每則最多 80 字。
