連線設定

填入 AI 的 API 金鑰與模型，卄珩才能真正回話。資料只存在你這台電腦的瀏覽器裡，不會上傳到任何地方。

小提醒：本檔直接雙擊用瀏覽器開啟即可使用，建議用 Chrome 或 Edge。若已確認金鑰與模型沒填錯，但「測試連線」仍一直失敗，多半是該瀏覽器擋下了 file:// 的對外連線——換成 Chrome 或 Edge 開啟通常就能解決。

人格設定

這裡決定卄珩「是誰、怎麼說話」。下面的設定會在每次對話時提供給 AI，讓他保持人設一致。

事件

上傳或貼上角色設定、世界觀、人物關係等資料。每筆事件最多 2000 字，最多 30 筆。這些內容會一起提供給 AI，讓卄珩更貼合設定。

記憶體

由角色依照每 10–20 則對話自動生成的記憶內容（標示「自動記憶」），越新的記憶越被重視。最多儲存 100 筆，你也可以手動新增、修改或刪除。每則最多 80 字。

生活日誌

由你自行新增與編輯的生活日誌，記錄每天的點滴。可以新增、修改或刪除每一頁。每則最多 80 字。
